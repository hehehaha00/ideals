项目遗迹博物馆

一句话

扫描用户废弃的 GitHub 仓库，把它们变成一个可浏览、可纪念、可提取残留价值的作品遗址。

为什么有趣

它反过来处理创作焦虑：不是催人继续，而是承认有些东西值得好好结束。

来源词
独立开发者深夜烂尾焦虑GitHub 仓库博物馆不能催促
第一版形态

一个本地网页工具：读取仓库列表、README 和最近提交，生成项目卡片、墓志铭、残留价值和可复活零件。
